MERN Discovery

Since the last post I (Conor) have completed half of MongoDB University's Intro to MongoDB course. This how now given me a very good foundational understanding of how MongoDB databases are created and structured. We have also discovered the modern and seemingly popular MERN structure for full-stack web development which is a structure we will most likely be using for this project. Each letter of MERN stands for a different language / technology. M for MongoDB, E for Express js, R for React js and N for Node js. React will obviously handle our front end, MongoDB will be our database and a combination of Node js and Express js will handle our backend.
